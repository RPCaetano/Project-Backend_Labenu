# Project-Backend_Labenu

Projeto desenvolvido em TS para apresentação de um projeto backend e projeto final do curso FULLSTACK da LABENU. Este projeto tem como objetivo cadastrar usuário coletando seus dados completos como nome, nickname,email e senha, após cadastro o usuário tem acesso as imagens e seus detalhes que são coletados através do seu ID. Mais detalhes visuais estão em https://github.com/RPCaetano/Project-Frontend-Labenu.

###Para rodar localmente: -Fazer git clone; -Realizar no terminal o comando npm i para as instalações; -Rodar como npm run start;****

### Deploy foi realizado Heroku.
